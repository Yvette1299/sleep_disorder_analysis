# 基于睡眠数据集的问题分析

### 项目概述｜Project Overview

睡眠是人类生理与心理功能恢复的基础环节。医学研究表明，成年人每日需7-9小时高质量睡眠以维持稳定的代谢健康、优化的神经内分泌调节，从而保持长期的身体机能。然而，在如今快节奏、高压力的现代社会，睡眠已从一种自然的生理需求，演变为一种稀缺的“奢侈品”。世界卫生组织已将睡眠问题视为全球性的公共卫生问题。长期的睡眠不足或睡眠质量低下，不仅仅会导致“休息不好”，影响工作与学习，更可能导致一系列严峻的身心健康问题，包括但不限于：心血管疾病、糖尿病、肥胖、抑郁症、焦虑症、免疫功能下降以及认知功能受损等等。

然而，在睡眠问题日益严重的当下，公众对睡眠健康的认知仍存在巨大缺口。许多人将打鼾、日间困倦视为常态，忽视了其背后可能隐藏的睡眠障碍（如睡眠呼吸暂停）。同时，不同职业、不同生活方式的人群，其睡眠模式与面临的挑战也截然不同。因此，本研究旨在利用机器学习技术，对包含多维度特征的睡眠健康数据集进行深入挖掘，通过三个具体的研究方向，揭示影响睡眠健康的关键因素，为不同职业人群改善睡眠质量、预防睡眠障碍提供针对性依据。

---

### 主要目的｜Main Purpose

1. 基于关键特征预测睡眠质量

2. 基于多维特征预测睡眠障碍类型

3. 探查被职业特性塑造的睡眠模式

---

### 输出结果｜Output Results
**01_sleep_quality.ipynb**

此文件研究了基于关键特征预测睡眠质量的问题，包含了从数据预处理，到利用线性回归模型进行睡眠质量预测，再到模型评估的全过程，最终效果中R²分数为0.89，表明模型能解释目标变量近90%的波动，拟合效果好，预测精度较高。

[../notebooks/01_sleep_quality.ipynb](../notebooks/01_sleep_quality.ipynb)

**02_sleep_disorder_type.ipynb**

此文件研究了基于多维特征预测睡眠障碍类型的问题，包含了对数据进行独热编码、标准化处理等操作，以及决策树模型的建立，以及随后的超参数调优的全过程。调优后的模型在测试集上取得了94.67%的准确率。在后续对具体的特征重要性分析中，我们发现血压的舒张压、BMI和压力水平与睡眠障碍类型有着较为紧密的关系，这也对睡眠障碍的预防和诊断提供了方向。

[../notebooks/02_sleep_disorder_type.ipynb](../notebooks/02_sleep_disorder_type.ipynb)

**03_occupation_and_sleep.ipynb**

此文件通过K-means聚类分析，挖掘不同职业人群在睡眠时长、焦虑水平、运动习惯等核心变量上的聚集规律，进而识别具有相似特征的睡眠健康亚群，为针对性干预提供依据。

文件包含数据预处理、绘制并分析特征关系热力图，并采用肘部法则和轮廓系数来确定最佳K值以进行K-means聚类，并根据聚类结果进行可视化分析，将研究对象（不同职业人群）划分为6个具有显著特征的聚类群体，通过数据探讨了了不同职业与健康的关联、生活方式对BMI的影响，并基于分析结果给出了维持健康的小建议。
